# Задание 5: выбор лидера
- Каждый узел — отдельный `Thread`.
- Входящие сообщения приходят в `BlockingQueue`.
- Лидер всегда один: живой узел с самым большим `id`.
- Сообщения: `PING`, `ELECT`, `ANSWER`, `VICTORY`.
- Узел можно уронить и поднять вручную (`down <id>`, `up <id>`).
